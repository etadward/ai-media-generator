# 概念先行 Prompting — 治「漂亮但空洞」

**這份檔案治一個病：prompt 堆滿技術詞（subsurface scattering / caustics / volumetric / anamorphic），畫面很炫，但「不知道在幹嘛」—— 沒有主題、沒有故事、鏡頭各自為政。**

> 技術詞 ≠ 好廣告。技術詞只決定「畫面漂不漂亮」，**概念**才決定「這支片在說什麼、賣什麼感覺、為什麼有人看完想買」。一支沒有 concept 的片，每個鏡頭都美，但加起來是零。

---

## ⛔ 最重要前提：concept ≠ 抽象化產品（2026-06-08 血淚修正）

**這份檔案第一版被我自己用錯了。** 我把「concept」做成抽象藝術 —— 「一滴的奢侈」變成「一滴金液懸浮純黑虛空」，結果產品（瓶子）幾乎看不到 = 用戶評「爛透了」。

**鐵則：concept 是「讓觀眾對這個『清楚可見』的產品產生渴望」，不是「把產品變成隱喻 / 藏進抽象光影」。**

- ✅ 產品**真實、認得出、完整清楚**出現在畫面（漢堡看得出漢堡、鞋看得出鞋）
- ✅ concept 服務「產品被看見 + 被渴望」（食慾、想要、想買）
- ❌ 不要抽象 VFX（粒子/裂縫/流體/純光影）當產品廣告
- ❌ 不要極簡到產品消失（「懸浮黑虛空」= 看不到產品 = 空洞）

**送出前 4 題自我把關**（見 memory `feedback_no_abstract_concrete_hero.md`）：
1. 產品清楚完整出現了嗎（還是躲抽象光影裡）？
2. 路人看 3 秒認得出這是什麼產品嗎？
3. 這像「廣告」還是像「炫 VFX 技術」？
4. concept 是讓產品更誘人，還是把產品抽象掉？
任一題答錯 → 重寫。

---

## 0. 自我診斷（寫 prompt 前先問）

寫任何影片 prompt 前，先用一句話回答：

> **「這支片在賣什麼『感覺』 / 說什麼『故事』 / 給觀眾什麼『一個』記憶點？」**

如果答不出來，或答案是「展示產品很高級」這種空話 → **停**。你還沒有 concept，現在寫的會是空洞鏡頭集。

**反面教材（本 skill 之前犯的）：**
```
[00:00-00:05] 精華液瓶 ultra-macro dolly-in，subsurface scattering，caustics 光紋…
[00:05-00:10] 金色液滴 crown splash，ray-traced 折射…
```
↑ 技術滿分，但 concept = 0。它只是「一個瓶子被拍得很漂亮」。觀眾看完不記得任何事。

**正面改寫（同產品，有 concept）：**
> **Concept：「一滴的奢侈」—— 整支片只為了讓你渴望那『一滴』金色精華，像渴望時間本身。**
```
[00:00-00:04] 純黑中，唯一光源照亮懸停的單一金色液滴，緩慢墜落 ——
              世界屏息。Slow push-in 跟著液滴。
[00:04-00:08] 液滴觸及瓶口的瞬間，瓶身才從黑暗中浮現（揭示 reveal），
              金色從滴落點向整瓶擴散點亮。Match cut。
[00:08-00:10] 拉開：瓶身定格在黑色基座上，一行光暈。
              Final beat：那滴金，是你每天值得的奢侈。
```
↑ 同樣的瓶子、同樣的微距，但現在**每個鏡頭都在服務「一滴的奢侈」這個 idea**。觀眾記得「那一滴」。

## 1. 概念先行的 4 步流程

### Step 1 — 一句話 Concept
用「[產品/主題] 是 [一個出乎意料的比喻/感覺/承諾]」造句。

| 弱（功能描述）| 強（概念）|
|---|---|
| 展示腕錶的精密機芯 | **「時間活著」** —— 機芯像心臟在跳，你戴的是一個活物 |
| 精華液很保濕 | **「一滴的奢侈」** —— 渴望那一滴，像渴望時間 |
| 跑鞋很輕 | **「重力的叛徒」** —— 穿上它，地心引力對你失效 |
| 咖啡很香 | **「第一口的甦醒」** —— 整個世界在你喝下第一口時才開機 |

### Step 2 — 3-Beat 敘事弧（每支片，哪怕 10 秒）
即使只有 10 秒，也要有「起 → 轉 → 收」：

- **Setup（起）**：建立一個張力 / 期待 / 問句。「這是什麼？」「接下來呢？」
- **Turn（轉）**：揭示 / 變化 / 高潮。產品登場、效果發生、真相浮現。
- **Payoff（收）**：定格 + 情緒落點。讓觀眾帶走「那個感覺」。

**10 秒分配：** Setup 0-4s｜Turn 4-8s｜Payoff 8-10s。

### Step 3 — 每個鏡頭做「存在性測試」
對每個 shot 問：**「這個鏡頭在說什麼？刪掉它，故事還成立嗎？」**
- 答得出它推進了 concept → 留。
- 只是「再來一個漂亮角度」→ **刪**。鏡頭少而準 > 多而空。

這直接治「有些畫面不知道在幹嘛」—— 那些畫面就是沒通過存在性測試的。

### Step 4 — 才加技術詞（服務概念，不是炫技）
概念與敘事定了，**最後**才補質感 token（光線/鏡頭/材質）。而且只補「服務這個 concept 的」：
- 「一滴的奢侈」→ 需要 `單一聚光、極暗背景、淺景深`（孤立那一滴）。不需要滿屏 caustics。
- 技術詞是形容詞，概念是主詞。**永遠先有主詞。**

## 2. 產品廣告的 6 個概念框架（卡住時直接套）

每個都是經過驗證的廣告母題。挑一個，把你的產品填進去：

| 框架 | 一句話 | 適合 | 範例 concept |
|---|---|---|---|
| **轉變 Transformation** | before → after，產品是轉折點 | 美妝/健身/清潔 | 「暗沉的早晨 → 一抹精華 → 發光的她」 |
| **英雄揭示 Hero Reveal** | 萬眾矚目中產品登場 | 3C/車/精品 | 「黑暗中，它一件一件點亮自己」 |
| **感官誘惑 Sensory** | 放大單一感官到極致 | 食飲/香氛 | 「你能聽見巧克力裂開的聲音」 |
| **擬人/賦予生命 Anthropomorphism** | 產品像有生命 | 腕錶/機械/科技 | 「機芯像心臟在跳，你戴著一個活物」 |
| **問題→解放 Problem→Relief** | 痛點 → 產品 → 鬆一口氣 | 工具/日用/服務 | 「纏結的線 → 一個動作 → 全部順了」 |
| **單一英雄物件 One Hero** | 整片只為一個元素 | 任何高端品 | 「一滴的奢侈」 |

**選框架法則：** 產品最想讓人記住的「那一件事」是什麼 → 對應框架。別貪心塞兩個。

## 3. 影片「效果差」的概念面根因（不只是模型問題）

「生出來效果差」很多時候不是模型爛，是 concept/分鏡的問題：

| 症狀 | 概念面根因 | 修法 |
|---|---|---|
| 畫面漂亮但無聊 | 沒 concept，只是美圖動起來 | Step 1 補概念 |
| 鏡頭之間沒關聯、跳 | 沒 3-beat 弧，鏡頭各自為政 | Step 2 串敘事 |
| 「不知道在看什麼」 | 主體不明確 / 視覺重點分散 | 每個 shot 鎖一個視覺焦點 |
| 平、沒情緒 | 沒有 Turn（轉折/高潮） | Step 2 加一個「揭示」或「變化」時刻 |
| 像 stock footage | 沒有獨特 idea，套路運鏡 | 換一個出乎意料的比喻當 concept |

> **先救 concept，再救技術。** 一個有 concept 的片用普通模型也動人；一個沒 concept 的片用 Veo 4K 也是貴的空洞。

## 4. 概念 → prompt 的接法（跨模型通用）

把 concept 翻成 prompt 時，**concept 寫在開頭當總綱**，再展開 beats：

```
[Concept / 一句總綱：這支片要傳達的「一個感覺」]
[00:00-00:0X] Setup —— {建立張力的鏡頭}。{運鏡}。Smooth transition to:
[00:0X-00:0Y] Turn —— {揭示/變化/高潮}。{運鏡}。Match cut to:
[00:0Y-00:10] Payoff —— {定格 + 情緒落點}。Final beat: {記憶點}。
[視覺風格 / 色彩 / 音訊 / Constraints —— 服務上面 concept 的，不堆無關炫技詞]
```

**模型差異**（細節見各 reference）：
- **Gemini Omni / Veo 3.1**：吃敘事，可寫情緒轉折、對白、世界邏輯 → concept 可寫深
- **Seedance 2.0**：多鏡頭敘事強，`@AssetName` 鎖主體 → concept + 3-beat 直接做
- **Hailuo 2.3**：「導演的 AI」，要劇本不要清單 → concept-first 天生契合
- **Kling 3.0**：序列動作（First…then…finally）比同時動作穩 → 3-beat 用序列寫
- **Vidu Q3**：`@tag` 多主體 + Smart Cuts 自動分鏡 → concept 給它，分鏡交給它

## 5. 反模式（別再犯）

- ❌ **技術詞堆疊當作品質**：subsurface scattering + caustics + volumetric + anamorphic 全塞，但沒 concept = 空洞。
- ❌ **鏡頭各自為政**：每個 shot 都漂亮但互不相關，沒有弧。
- ❌ **為炫技而炫技**：加一個 360° orbit「因為很酷」，但它不推進任何 idea。
- ❌ **概念是空話**：「展現高級感 / 質感滿滿」不是 concept，是形容詞。Concept 要具體到能當片名。
- ❌ **貪心塞多概念**：一支 10 秒片塞「轉變 + 英雄揭示 + 感官」= 全部都淺。一片一概念。

---

## 6. 一句話心法

> **先想「這支片讓人記住哪一件事」，再想「怎麼拍漂亮」。順序顛倒 = 漂亮的空洞。**
